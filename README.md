# RPGGUI

Uma interface de usuário para RPGs simples, desenvolvida em Python, focada em gerenciamento de personagens e alteração de status ao longo dos turnos.

O objetivo do projeto é fornecer uma base para manipulação de personagens em sistemas de RPG, permitindo criar, armazenar e modificar atributos de forma organizada, baseado em um sistema de RPG montado por um amigo. Servindo como estudo para a criação de aplicativos e utilização de tecnologias como Git e Github.

# 🧩 Funcionalidades planejadas

- Criação de personagens

- Sistema de atributos

- Atualização de status durante turnos

- Armazenamento de dados em JSON

- Interface gráfica simples

- Sistema de inventário

- Sistema de rolagem de dados
